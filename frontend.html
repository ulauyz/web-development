Front-end


Front-end (клієнтська частина) охоплює ті технології та інструменти, які використовуються для створення користувацького інтерфейсу веб-додатка. Це та частина, яку користувач бачить та взаємодіє з нею. Основні складові front-end розробки включають мови програмування, фреймворки, бібліотеки, інструменти для розробки та тестування.

HTML

Мова розмітки, яка використовується для створення структури та вмісту веб-сторінок.

CSS

Мова стилізації, яка використовується для визначення вигляду та розміщення елементів на веб-сторінці.

JAVA SCRIPT

Мова програмування для динамічної інтеракції на веб-сторінці, валідації форм, анімацій та інших функцій.

REACT JS

Бібліотека js для створення інтерфейсів користувача, яка вирішує проблеми часткового оновлення вмісту вебсторінки

ANGULAR

Фреймворк дозволяє створювати динамічні та односторінкові додатки за допомогою шаблонів, об'єктів та інше.

BOOTSTRAP

Фреймворк пропонує набір готових компонентів, шаблонів та стилів, які спрощують процес розробки..

jQUERY

Бібліотека js, яка спрощує взаємодію з HTML-документами, обробку подій, анімацію та взаємодію з сервером за допомогою AJAX.

MOCHA

Фреймворк для тестування коду на JS. Mocha створює та виконує тести для JS-коду, незалежно від того, чи це клієнтський, чи серверний код.

JEST

Фреймворк для тестування JS-коду,який використовується у веб-додатках, де використовується React, Vue, Angula.
