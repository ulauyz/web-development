Back-end


Back-end в інформаційних технологіях відповідає за обробку та збереження даних, а також виконання бізнес-логіки для веб-додатків. Він забезпечує взаємодію між фронтендом (клієнтською частиною) та сервером, відповідаючи за виконання запитів користувачів та надання їм відповідей.

JAVA

Мова широко використовується для розробки різноманітних додатків, від мобільних додатків до веб-програм та великих корпоративних систем.

PYTHON

Високорівнева, об'єктно-орієнтована мова програмування, створена Гвідо ван Россумом.Вирізняється простотою синтаксису та читабельністю коду.

NODE.JS

Мова для створення серверів та мережевих додатків, що дозволяє використовувати JS для розробки на стороні клієнта і на стороні сервера.

RUBY

Об'єктно-орієнтована мова програмування, яка була створена Юкіхіро Мацумото у Японії в 1995.

EXPRESS

Фреймворк дозволяє створювати серверні додатки, забезпечуючи зручний інтерфейс для обробки HTTP-запитів та відправки відповідей.

DJANGO

Фреймворк включає ряд інструментів,які полегшують створення веб-додатків та забезпечення їх ефективного функціонування.

JSON Web Tokens

Використовується для обміну інформації між сторонами та для автентифікації та передачі відомостей про користувача між сторонами.

REST API

Стиль архітектури для побудови веб-служб, який дає змогу веб-додаткам обмінюватися даними.

MYSQL

Управління базами даних, яке базується на мові SQL- мови запитів, яка використовується для видалення, оновлення та вибору даних в базі.
